# ボートレースAI統合システム GAS台帳

## 目的

ボートレースAI統合システムで使用するGoogle Apps Scriptの
現行コード、変更履歴、復旧手順を管理する。

## 正本の扱い

- 現在の本番稼働コード：Google Apps Script
- バックアップ・変更履歴：このGitHubリポジトリ
- Google Drive上の「GAS台帳0606」は旧資料
- 旧Phase1βコードを現行仕様として使用しない
- 現行仕様は2026-07-19引き継ぎ設計書と最新本番GASを優先する

## 基本ルール

1. 本番GASを変更する前にGitHubへ保存する
2. 変更理由をコミットメッセージに書く
3. 動作確認後にバージョンタグを付ける
4. APIキー、パスワード、Webhook URLは保存しない
5. 廃止コードは削除せずarchiveへ移動する
